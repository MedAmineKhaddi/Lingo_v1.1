Project Title: Lingo - Language Learning Platform

Description:
Lingo is an interactive language learning platform similar to Duolingo. It provides users with an engaging and gamified environment to learn new languages. Users can track their progress, practice skills, and improve their language proficiency.

Technologies Used:
- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- ESLint

Installation Instructions:
1. Clone the repository:
   git clone https://github.com/username/lingo.git
2. Navigate to the project directory:
   cd lingo
3. Install the required dependencies:
   npm install
4. Set up the environment variables (create a `.env` file in the root directory if needed).
5. Run the development server:
   npm run dev
6. Open your browser and navigate to `http://localhost:3000` to start using Lingo.

Usage:
1. Open the application in your browser.
2. Register or log in to your account.
3. Select a language to start practicing.
4. Follow the lessons and complete exercises.
5. Track your progress and earn rewards for completing challenges.

Features:
- Language learning courses and exercises.
- Progress tracking and achievements.
- Interactive quizzes and challenges.
- Responsive design for mobile and desktop users.
- User authentication with Clerk middleware.
